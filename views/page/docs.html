The Build Tools

This framework is build using AngularJS 1.2 with Bootstrap 4 / UI.Bootstrap strapped.

It uses GruntJS Javascript Task Runner for various tasks such as organising CSS and JavaScript as well as running a browser-sync server.

  1. Run $ grunt to start server with browserSync & watcher

Note: Command must be run from the project's root folder. If for any reason you're experiencing difficulties, attempt running: $ npm install from the root directory to install dependencies, then try running $ grunt again.

CSS Bootstrap / Custom / Overides / Policy (BEM)

This framework uses GruntJS Javascript Task Runner for its CSS and JavaScript build system.

  1. Download and install Node, which we use to manage our dependencies.
    (The Webpack days....Maybe something for the future ;)
  2. Run npm install -g bower grunt-cli
  3. Run bower install --force-latest to install dependencies
  4. Run npm install to install dependencies
  5. Run grunt gojavelin to start server with browserSync & watcher

Note: All commands must be run from the project's root folder.

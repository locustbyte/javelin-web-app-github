Placing elements in the styleguide

When placing UI elements on the page you should follow the following format below. This format will place the element in a 'Box' with a header and body which you can use to label and place the code respectively.


                        <div class="box__jls-variant1" id="ui-search__[pattern reference]">
                            <div class="box__header__brand-neutral-light">
                                [Pattern Reference] <code>[ANY VARIANT ATTRIBUTES]</code>
                                <small>Pattern ref: <code class="grey-900">[pattern reference]</code></small>
                            </div>
                            <div class="box-body">
                                <!--begin pattern -->
                                <div class="makecodeview">
                                    [YOUR PATTERN CODE HERE]
                                </div>
                                <!--end pattern -->
                                <pre ng-copyable class="text-left" data-toggle="modal" data-target="#bottom"></pre>
                                <span class="clip__code label label-md accent pos-rlt m-r-xs"><b class="arrow top b-accent pull-left"></b>Click Code to Copy</span>
                            </div>
                        </div>
Click Code to Copy

NOTE: Be sure to change in the code below where you see the [ xxxxx ] as this indicates code you need to change to your specific pattern code / reference.

List New Pattern in 'Pattern Search' feature

In order to get your pattern listen in the Pattern Search Feature, you will need to:

Step One

Ensure than you have appended to the first line of the 'box' element that is displaying your pattern, has an ID which is prefixed with ui-search__ followed by your pattern reference as shown below.

                            <div class="box__jls-variant1" id="ui-search__my-pattern-reference">
Step Two

Locate the patterns.json file which is located in /app/scripts/controllers/ and enter a new row (might be easier to copy paste a row already existing) and use common sense by reading the key pairs what data you should enter.

You're done!
KDiff3 Git Merge Tool
  • 
    Download Link (WIN & PC)
    https://sourceforge.net/projects/kdiff3/files/kdiff3/0.9.98/
  • 
    Installation Instructions MAC OS X
    • Step One Open .DMG file and move kdiff3 app to /Applications
    • Step Two $ git config --global merge.tool kdiff3
    • Step Three $ git config --global mergetool.kdiff3.path /Applications/kdiff3.app/Contents/MacOS/kdiff3
  • 
    Installation Instructions PC
    • Step One Run .exe file and note installation path
    • Step Two git config --global merge.tool kdiff3
    • Step Three git config --global mergetool.kdiff3.cmd '"C:\\Program Files (x86)\\KDiff3\\kdiff3" $BASE $LOCAL $REMOTE -o $MERGED'
